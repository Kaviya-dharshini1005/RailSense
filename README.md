# RailSense 🚆
Autonomous AI Track Inspection Agent for Railways

## Problem
Railway trackmen manually inspect 10-15km of track daily on foot — 
a dangerous 19th century method causing 55.8% of train accidents 
to involve staff failures. Dedicated inspection trains like Japan's 
Doctor Yellow cost ₹500+ crore and work only on premium networks.

## Solution
RailSense mounts on any existing maintenance trolley and autonomously 
inspects tracks using computer vision and multi-agent AI — 
zero humans on track, zero dedicated infrastructure needed.

## Agents
- Agent 1: Vision Inspector — detects cracks, loose fasteners, geometry issues
- Agent 2: Prioritisation Agent — classifies severity autonomously  
- Agent 3: Reporting Agent — generates structured maintenance reports
